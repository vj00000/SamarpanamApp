# PranaFlow Yoga

A comprehensive yoga assisting app offering guided pranayama techniques, video tutorials, wellness blogs, and Ayurvedic food recommendations. Track your progress and save your favorite practices for a personalized yoga journey.

## Design & Contribution

For developers looking to contribute or understand the project's visual identity, please refer to our [Style Guide](./STYLE_GUIDE.md).

## Features

1. **Pranayama Techniques**
   - Guided breathing exercises with step-by-step instructions
   - Duration and difficulty indicators
   - Favorites functionality to save preferred techniques

2. **Video Guides**
   - High-quality instructional videos for various yoga practices
   - Playback controls integrated directly in the app

3. **Wellness Blog**
   - Articles on yoga philosophy, benefits, and lifestyle tips
   - Ayurvedic wisdom and holistic health practices

4. **Aahar (Food) Recommendations**
   - Ayurvedic dietary suggestions based on body types
   - Seasonal recipes and nutritional guidance

5. **Progress Tracking**
   - Practice streaks and consistency metrics
   - Detailed history of completed sessions
   - Personalized insights and achievements

6. **Favorites System**
   - Save preferred techniques, articles, and recipes
   - Quick access to personal collection

## Technologies Used

- **React Native**: Cross-platform mobile development framework
- **Expo**: Development platform for universal React applications
- **NativeWind**: Tailwind CSS implementation for React Native
- **Lucide Icons**: Clean and consistent iconography
- **React Navigation**: Intuitive navigation between app sections

## Installation Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/vj00000/SamarpanamApp.git
   ```

2. Navigate to the project directory:
   ```bash
   cd pranaflow-yoga
   ```

3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

4. Start the development server:
   ```bash
   npx expo start
   ```

5. Run on iOS simulator:
   ```bash
   npx expo run:ios
   ```

6. Run on Android emulator:
   ```bash
   npx expo run:android
   ```

## Usage Instructions

1. Launch the app on your preferred device or simulator
2. Browse through the main categories: Pranayama, Asanas, Meditation, and Aahar
3. Explore featured techniques and mark favorites by tapping the heart icon
4. Start practicing by selecting any technique and following the guided instructions
5. Track your progress in the "Your Progress" section
6. Read wellness articles and discover Ayurvedic recipes in the resources section
7. View your saved favorites for quick access to preferred content

The app provides a seamless experience for practitioners of all levels, combining traditional wisdom with modern technology to support your wellness journey.
